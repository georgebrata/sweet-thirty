## How to install
1. Clone this repositoy:

```bash
git clone https://github.com/georgebrata/sweet-thirty.git
```

2. Go into project

```bash
cd sweet-thirty
```

3. Install dependencies

```bash
npm install
```

4. Run project

```bash
npm run dev
```
